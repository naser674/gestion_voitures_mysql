## Description
Ce projet Python a pour objectif de gérer une base de données MySQL contenant des informations sur des voitures.

## Fonctionnalités
- connexion à MySQL
- utilisation d’un fichier JSON pour la configuration
- ajout d’une voiture
- affichage des voitures
- modification d’une voiture
- suppression d’une voiture

## Technologies utilisées

- MySQL
